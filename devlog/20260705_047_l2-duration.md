# 20260705_047 — L2 지속시간 게이트

> 정합성 게이트에 L2 추가. L1(authored order)은 *sparse* — 사람이 놓은 쌍만 검사.
> L2 는 게이트웨이가 덮는 **모든 유닛의 duration 을 자동 검사**하는 파생 안전망.

## 한 일 (`engine/evaluate.py::_certify`)
- 게이트웨이 산출값 → 유닛 base. **rank 별 타일링**으로 각 유닛 duration = base − top(이전 젊은 base, 최연소=0).
- `duration ≤ 0` (같은 rank 인접 base 동일/역전 = 영-길이 유닛) → **L2 fail**, `passed=False`. 없으면 pass.
- 게이트웨이 없으면 skip. 공유 경계(base-triassic ≡ base-induan)는 *다른 rank* 라 rank 별 검사에서 충돌 없음.

## L1 과의 상보성 (핵심)
- order 노드는 `gap ≥ min_gap(0)` — **coincidence(gap=0) 허용**. 그래서 두 경계가 같아져도 L1 pass.
- L2 는 유닛 **duration > 0** 을 요구 → 영-길이 유닛을 잡음. 검증: pub-olenekian 을 anisian 과 동일화하면
  **L1 pass · L2 fail**. authored order 가 안 덮는 곳도 L2 가 자동으로 커버.

## 검증
- clean seed: `{L0 pass, L1 pass, L2 pass, L3 skip}` passed. 3 예제·조립 그래프 모두 L2 pass.
- `pytest` **81 passed**(신규 L2 테스트: clean pass, 영-길이 → L1 pass·L2 fail). 마이그레이션·재시드·프론트 변경 없음.

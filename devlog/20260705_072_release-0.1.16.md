# 20260705_072 — 릴리스 0.1.16 (선택 박스 Full 모드)

## 0.1.16 에 담긴 것

- **선택 박스가 드래그 범위를 넘던 문제** — 놓는 순간 나타나는 "선택 노드 bounding box" 의 x2 가
  커서보다 오른쪽 고정 위치로 튐. 원인: `SelectionMode.Partial` 은 rubber-band 가 **살짝 스친**
  넓은 노드(process/data)까지 선택 → box 가 그 노드 오른쪽 끝까지 확장.
  `SelectionMode.Full` 로 변경 — **완전히 감싼 노드만** 선택 → bounding box 가 그린 사각형을 못 넘는다
  (표준 마퀴). 프론트 전용.

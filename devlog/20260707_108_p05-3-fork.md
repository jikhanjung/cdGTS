# 20260707_108 — P05.3: Fork

[P05](20260707_P05_arc-c-multiuser-ci-platform.md) 3단계. 읽을 수 있는 그래프를 **깊은 복제**해 내 샌드박스로.
P05.2로 데모/남의 그래프가 읽기 전용이 됐으니, fork가 소유 사본을 얻는 경로.

## 백엔드

- **`Graph.forked_from`** self-FK(lineage). migration graph.0008.
- **`graph/services.py:fork_graph(source, user)`** — 트랜잭션 깊은 복제: groups(→parent→lower/upper 순) ·
  nodes(key 보존) · edges · gateways 재링크(key로). 새 Graph(owner=user, status=sandbox, forked_from=source,
  고유 slug `<src>-fork[-n]`).
- **`POST /api/graphs/{id}/fork/`**(action) — 인증 필요, `get_object`이 가시성 강제(볼 수 있는 것만 fork).
  `GraphAccessPermission`에 fork 예외(원본 미변경 → owner 아니어도 인증만 되면 허용). 201 + 새 그래프 반환.
- 직렬화에 `forked_from` 노출.

## 프론트

- `forkGraph(id)` API. Editor에 **Fork 버튼**(로그인 시): 비-owner(데모/남의 것) 볼 땐 primary 강조.
  클릭 → fork → 목록 추가 + 즉시 그 그래프로 전환(이제 owner=나 → 편집·Bake 가능).

# 20260703_030 — 릴리스 Diff UI

> 미뤄뒀던 diff UI 추가. 백엔드 `diff_releases`(값/토폴로지 직교)는 이미 있었고, 프론트만 붙였다.

## 한 일

### 프론트엔드 (React)
- `App.jsx`: 상단 nav 로 **에디터 / 릴리스 Diff** 뷰 전환. `ReactFlowProvider` 는 Editor 만 감쌈.
- `ReleasesDiff.jsx`(신규): 릴리스 A→B 선택 → `GET /api/releases/diff/?a=&b=` 호출.
  - **값 diff** 표(경계·A·B·Δ Myr, 증감 색), **토폴로지 diff** 리스트(added/removed/retype 뱃지).
  - bake 안 된 릴리스면 인라인 bake 버튼. 릴리스 0개면 빈 상태 안내.
- `Editor.jsx`: 툴바에 **그래프 선택 드롭다운** 추가(예제 그래프 열람용). `loadGraph(id)` 헬퍼.
- `api.js`: `listReleases`·`getRelease`·`bakeRelease`·`diffReleases`. `index.css`: nav/diff 스타일.

### 시드 (`releases/fixtures/example_releases.json`, 신규)
- Diff 데모용 두 릴리스 + 후보/출력/선택/레코드 완결 체인:
  - `ICC-2012 (예시)`: base-triassic 252.5±0.3, base-cambrian 542.0±0.3 (재보정 이전).
  - `ICC-2024 (예시)`: base-triassic 251.902±0.024, base-cambrian 538.8±0.6, **+ base-proterozoic 2500(신규)**.
- Diff(2012→2024): 값 diff 2건(−0.598 / −3.2 Myr), 토폴로지 diff 1건(base-proterozoic **added**).
  문헌 서사(재보정으로 숫자 이동)를 그대로 보여줌.

## 검증
- `example_releases` loaddata 22객체. diff API = 위 값 정확히 반환. bake 재실행 → 동일 레코드(일관).
- `npm run build` 성공(192 modules, 에러 없음). 백엔드 `pytest` 59 passed.

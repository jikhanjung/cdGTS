# cdGTS Quick Start — Example ④를 fork 해서 한번 만져보기

*[English](quickstart_en.md) · 한국어*

처음 오신 분을 위한 5분짜리 실습입니다. 앱을 열고 → 로그인 → 가장 큰 예제 그래프를 **fork** →
값 하나를 바꿔 → 그 변화가 경계 연대에 어떻게 퍼지는지 봅니다. cdGTS가 무엇인지는
[소개글](introduction.md)을 먼저 보셔도 좋습니다.

> **미리 알아둘 것.** 계정은 **초대제**입니다(공개 가입 없음) — 받으신 아이디로 로그인하세요.
> 계정이 없어도 **읽기 전용으로 둘러보는 것**은 됩니다. `fork`·편집·저장만 로그인이 필요합니다.

## 0. 앱 열기

브라우저로 **<https://cdgts.paleobytes.info>** 접속. 상단 내비게이션에 **Editor · Vault · Proposals ·
Bibliography** 탭이 있습니다. 처음엔 **Editor** 화면입니다.

## 1. 로그인

우측 상단 **Login** → 사용자명·비밀번호 입력 → **Sign in**. 로그인되면 상단에 사용자명이 뜹니다
(staff면 ★). *계정이 없으면 이 단계는 건너뛰고 3번까지 읽기 전용으로 따라와도 됩니다.*

## 2. Example ④ 열기

Editor 왼쪽 위 **그래프 선택 드롭다운**에서
**"Example ④: Partial ICC — assembling 3 boundaries (P–T · Cambrian base · Precambrian GSSA)"** 를 고릅니다.

이 그래프는 세 경계 — **페름–트라이아스**(GSSP·Meishan) · **캄브리아기 base**(대륙 간 상관) ·
**선캄브리아 GSSA**(약속된 숫자) — 를 완전한 provenance로 조립한 가장 큰 예제입니다(노드 ~280개).

## 3. 둘러보기 (읽기 전용)

- 로드되면 **자동 평가**가 돌아 각 노드에 결과가 붙습니다. **경계 노드(◈)** 얼굴에 연대(Ma)가 보입니다.
- **줌·팬**: Ctrl/⌘+휠로 확대, 드래그로 이동. 노드를 클릭하면 우측 **인스펙터**에 파라미터가 뜹니다.
- 시스템 그래프라 **🔒 Read-only** 배지가 보입니다 — 여기선 아직 못 고칩니다.

## 4. Fork 하기 (내 것으로 복사)

툴바의 **`Fork to edit →`** 버튼(또는 **Actions ▾ → Fork…**)을 누릅니다. 이름을 확인(기본
`… (fork)`)하고 확정하면 **"Forked → … · yours to edit"** 상태가 뜨고, 편집 가능한 **내 사본**으로
전환됩니다. 이제 **Save**가 활성화됩니다. *(로그인 안 했으면 "Sign in to fork"로 바뀝니다.)*

## 5. 값 하나 바꾸기

1. 경계 하나(예: **Base of Triassic**)로 들어가는 **데이터 leaf**(인용된 관측 — 방사연대 U–Pb 나
   공표값 같은 파란 노드)를 클릭.
2. 우측 **인스펙터**에서 값(예: 분포의 **`value_ma`**)을 다른 숫자로 수정.
3. 상단 상태가 **● Unsaved**로 바뀝니다.

## 6. 저장하고 결과 보기

- **Save** 클릭 → **✓ Saved**.
- **Actions ▾ → Evaluate** — 그래프를 다시 계산해 결과를 붙입니다. 경계 연대와 하단 결과가
  새 값으로 갱신됩니다. *(Evaluate·Verify는 **저장된** 그래프에 돌아가니, 편집 후 꼭 **먼저 Save**.)*

## 7. 공표본과 원클릭 diff ★

**Actions ▾ → Verify vs published** 를 누르면, 방금 바꾼 그래프를 공표 baseline(ICC `ICS-2024/12`)과
비교해 한 줄로 요약합니다:

> `vs published: N moved · max |Δ| X Ma · wiring ＋A/－R/↺T`

= 경계 **N개가 움직였고**, 최대 변화 **X Ma**, 배선(추가/삭제/retype) 요약. 이게 cdGTS의 핵심
루프입니다 — *새 근거를 넣으면 무엇이 바뀌나*를 손계산이 아니라 재현 가능한 산출로.

## 8. (선택) Bake 해서 Vault 에서 보기

**Actions ▾ → Bake…** 로 현재 결과를 **불변 릴리스**로 얼립니다. **Vault** 탭으로 가면 그 릴리스를
**ICC 차트·표·서술**로 보고, 다른 릴리스와 **Diff** 할 수 있습니다.

---

방금 한 일 = cdGTS의 두 축을 그대로 밟은 것입니다 — **엔진**(값을 바꾸면 경계가 재계산) +
**버전 관리**(얼린 릴리스 diff). 더 알고 싶으면:

- [소개글](introduction.md) — cdGTS가 무엇이고 왜 만드는가.
- [개념 지도](concept-map.md) — 티어·카테고리·문서 전체 지도.
- **되돌리기**: fork 는 **내 사본**이라 원본 시스템 그래프엔 영향이 없습니다. 마음껏 실험하세요.
